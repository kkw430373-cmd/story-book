# 📚 K-Story Readers

**한국 문화를 영어로 배우는 단계별 스토리북 앱**

"Step into Reading" 스타일의 5단계 영어 리딩 프로그램으로, 한국의 음식, 명절, 역사, 자연, 인물 등 다양한 문화를 영어 스토리북으로 학습할 수 있습니다.

---

## ✅ 현재 구현된 기능

### 핵심 기능
- **📊 구글 스프레드시트 실시간 연동**: 시트에 데이터를 추가하면 앱에 자동 반영 (5분 캐시)
- **🔍 5단계(Step 1~5) 필터링**: Step into Reading 스타일의 단계별 분류
- **🏷️ 카테고리 필터링**: 음식, 명절, 역사, 자연, 인물, 문화 등 주제별 분류
- **🔎 검색 기능**: 제목(영문/한글), 설명, 카테고리를 검색
- **📱 반응형 디자인**: 모바일, 태블릿, 데스크톱 지원
- **🔊 TTS(Text-to-Speech)**: 영어 제목 듣기 기능 (Web Speech API)
- **📖 스토리 상세 모달**: 카드 클릭 시 상세 정보 및 읽기 링크 제공
- **🔗 외부 링크 열기**: 잼스(GEMS) 스토리북을 새 탭에서 열기
- **💾 로컬 캐시**: 반복 방문 시 빠른 로딩 (5분 캐시)
- **⚡ 로딩 화면**: 데이터 로딩 중 애니메이션

### UI/UX
- 단계별 고유 색상 시스템 (Step 1: 초록, Step 2: 파랑, Step 3: 주황, Step 4: 보라, Step 5: 빨강)
- 썸네일 없는 경우 단계별 기본 그라디언트 + 이모지 자동 표시
- 카드 호버 애니메이션, 부드러운 모달 트랜지션
- 세미틱 HTML, 접근성(ARIA) 고려

```
/
├── index.html          # 메인 HTML 페이지
├── css/
│   └── style.css       # 전체 스타일 (반응형 포함)

│   └── app.js          # 메인 앱 컨트롤러
└── README.md           # 이 파일
```

## 📖 5단계 시스템

| 단계 | 이름 | 대상 | 특징 |
|------|------|------|------|
| Step 1 🌱 | Ready to Read | 유치원~초등 저학년 | 큰 글자, 쉬운 단어, 리듬감 |
| Step 2 🌿 | Reading with Help | 초등 저학년 | 짧은 문장, 단순한 이야기 |
| Step 3 🌳 | Reading on Your Own | 초등 1~3학년 | 캐릭터 중심, 매력적 줄거리 |
| Step 4 🏔️ | Reading Paragraphs | 초등 중학년 | 어려운 어휘, 문단 구조 |
| Step 5 🚀 | Ready for Chapters | 초등 고학년 | 챕터북, 긴 문단 |

## 🛠️ 기술 스택

- **HTML5 / CSS3 / Vanilla JavaScript** (프레임워크 없음)
- **Google Fonts**: Nunito, Noto Sans KR
- **Font Awesome 6.5**: 아이콘
- **Google Sheets CSV Export API**: 데이터 소스
- **Web Speech API**: TTS 기능
- **LocalStorage**: 클라이언트 캐싱

## 🚀 콘텐츠 추가 방법

1. **구글 잼스(GEMS)로 스토리북 생성**
2. **구글 스프레드시트**의 `스토리북` 시트에 새 행 추가
3. `단계`, `제목`, `제목_한글`, `링크 주소`, `카테고리`, `설명` 입력
4. 앱에서 **5분 후 자동 갱신** (또는 브라우저 캐시 삭제 후 새로고침)
